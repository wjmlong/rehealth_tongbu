## TOP 5 最优方案

### 方案1: 区县CVD风险智能热力图

**目标**: 为每个区县生成实时CVD风险分布地图，支持精准资源配置

**政绩亮点**:
- 可视化展示各区县健康差异
- 支持"健康浙江"考核数据上报
- 为区县卫健委主任提供决策依据

**数据使用**:
- 高血脂一览: 31,016人 × 8个区县
- 指标: SBP、TC、BMI、10年危险性
- 粒度: 街道 → 社区团队 → 个人

**PIAS能力**:
- CatBoost 16特征风险预测
- 区域聚合统计
- ECharts热力图配置

**输出**:
- API: `POST /api/pias/v2/predict` (批量)
- 可视化: 区县风险热力图 + 街道排名

---

### 方案2: 高血脂分级管理效果量化评估

**目标**: 用PSM+DRE方法证明高血脂分级管理（一级/二级/三级）的实际效果

**政绩亮点**:
- 用统计方法证明管理效果（p值、置信区间）
- 为"慢病管理示范区"申报提供数据支撑
- 可推广到其他慢病管理

**数据使用**:
- 2014→2015纵向数据: 25,583人
- 干预指标: 血脂异常分级随访管理（一级/二级/三级）
- 结局指标: TC、LDL-C、TG变化

**PIAS能力**:
- GroupAttributor: PSM + DRE + Bootstrap
- 完整统计检验: p值、E-value、功效分析
- 结算报告模板

**输出**:
- API: `POST /api/pias/v2/attribute/group`
- 报告: 管理效果量化报告（精算师级别）

---

### 方案3: 极高危人群靶向干预追踪系统

**目标**: 对10年CVD风险>20%的极高危人群进行精准干预追踪

**政绩亮点**:
- 精准健康管理典型案例
- 降低心血管事件发生率
- 可申报"智慧健康"创新项目

**数据使用**:
- 高血脂一览: 10年危险性>20%的人群
- 追踪指标: SBP、TC、LDL-C变化
- 干预措施: 用药、生活方式、随访频率

**PIAS能力**:
- IndividualAttributor: 个人风险轨迹预测
- 指数衰减模型 + 置信区间
- 个性化健康建议生成

**输出**:
- API: `POST /api/pias/v2/attribute/individual`
- 报告: 个人健康改善报告（用户版）

---

### 方案4: 社区卫生服务中心绩效智能排名

**目标**: 为每个社区卫生服务中心生成基于数据的绩效排名

**政绩亮点**:
- 客观、透明的绩效评估
- 支持基层医疗机构考核
- 激励先进、鞭策后进

**数据使用**:
- 高血脂一览: 69个社区团队
- 指标: 高血压控制率、血脂达标率、随访完成率
- 对比: 同区域、同类型中心

**PIAS能力**:
- 群体归因: 中心层面的干预效果
- 平衡性诊断: 排除患者基线差异
- 效应量: 实际改善幅度

**输出**:
- API: `POST /api/pias/v2/attribute/group` (按中心分组)
- 报告: 中心绩效排名 + 改进建议

---

### 方案5: 健康中国2030考核智能达标系统

**目标**: 自动生成符合"健康中国2030"考核要求的指标报告

**政绩亮点**:
- 自动化考核数据生成
- 减少人工统计工作量
- 提高数据准确性和及时性

**数据使用**:
- 两个数据集: 全量56,599条记录
- 考核指标: 高血压管理率、血脂异常管理率、吸烟率、BMI达标率
- 时间趋势: 2014→2015改善情况

**PIAS能力**:
- 风险预测: 基线风险评估
- 群体归因: 干预效果量化
- 分层报告: 监管机构版本

**输出**:
- API: 组合调用多个端点
- 报告: 考核达标报告（监管版）
